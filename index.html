üìù Git Commit Commands

git init

Creates a new Git repository.

Initializes a new Git repository in the current directory. Example: git init in a project folder to start version control.
git status

Show file status.

Displays the current state of the working directory and staging area. Example: git status shows modified files like index.html
git add file

Stage file changes.

Adds specified file changes to the staging area for the next commit. Example: git add index.html stages index.html for commit.
git commit -m "message"

Commit with message.

Records staged changes to the repository with a descriptive message. Example: git commit -m "Add homepage layout" commits changes.
git commit -a

Commit all changes.

Stages and commits all modified tracked files in one step. Example: git commit -a -m "Update all files" commits all modified files.
git commit --amend

Amend last commit.

Modifies the most recent commit with new changes or message. Example: git commit --amend -m "Fix typo in commit message" updates the last commit.

üìå Git Branch Commands

git branch

Display a list of the local branches.

Lists all local branches in the repository. Example: git branch lists branches like main, feature-x
git branch -a

List both local and remote branches.

Shows all branches, including local and remote-tracking branches. Example: git branch -a shows remotes/origin/main
git branch -c old new

Copy a Git branch.

Creates a new branch by copying an existing one. Example: git branch -c main feature-copy creates feature-copy from main
git branch -d branch-name

Delete a local branch (safe, requires merge).

Safely deletes a local branch if it has been merged. Example: git branch -d feature-x deletes merged feature-x
git branch -D branch-name

Force delete a branch (ignores merge state).

Forcefully deletes a local branch, even if unmerged. Example: git branch -D feature-x deletes feature-x regardless of merge.
git branch -m old-name new-name

Rename a branch.

Renames a local branch to a new name. Example: git branch -m dev development renames dev to development
git branch -r

List only remote branches.

Lists all remote-tracking branches in the repository. Example: git branch -r lists origin/main, origin/dev
git push <remote> --delete <branch>

Delete a remote branch.

Removes a branch from the remote repository. Example: git push origin --delete feature-x deletes feature-x on remote.
git push --set-upstream origin branch

Set upstream branch for pushing/pulling.

Links a local branch to a remote branch for tracking. Example: git push --set-upstream origin feature-x links local branch to remote.

üîÄ Git Checkout Commands

git checkout branch

Switch to a branch.

Switches the working directory to the specified branch. Example: git checkout dev switches to dev branch.
git checkout -b new-branch

Create and switch to a new branch.

Creates a new branch and switches to it immediately. Example: git checkout -b feature-x creates and switches to feature-x
git checkout commit-hash

Checkout previous commit (detached HEAD).

Checks out a specific commit, entering a detached HEAD state. Example: git checkout 1a2b3c4 views commit with hash 1a2b3c4
git checkout tag-name

Checkout a tag in a detached HEAD state.

Switches to a specific tag, entering a detached HEAD state. Example: git checkout v1.0 views tag v1.0

üìÇ Git Clone Commands

git clone repo-url

Clone a repository.

Copies a remote repository to the local machine. Example: git clone https://github.com/user/repo.git clones repo
git clone repo-url dir-name

Clone into custom folder.

Clones a repository into a specified directory. Example: git clone https://github.com/user/repo.git my-project clones into my-project
git clone repo-url --origin custom

Clone with custom remote name.

Clones a repository and sets a custom name for the remote. Example: git clone https://github.com/user/repo.git --origin upstream sets remote as upstream
git clone repo-url --branch branch

Clone and checkout specific branch.

Clones a repository and checks out a specific branch. Example: git clone https://github.com/user/repo.git --branch dev clones dev branch.
git clone repo-url --depth 1

Shallow clone with limited commits.

Clones a repository with only the latest commit history. Example: git clone https://github.com/user/repo.git --depth 1 clones latest commit only.

‚öôÔ∏è Git Config Commands

git config --global user.name "Name"

Set global username.

Sets the username for all repositories globally. Example: git config --global user.name "Dinesh Chowdary" sets username.
git config --global user.email "Email"

Set global email.

Sets the email address for all repositories globally. Example: git config --global user.email "Dinesh@example.com" sets email.
git config --list

Show config list.

Displays all current Git configuration settings. Example: git config --list displays settings like user.name=Dinesh Chowdary

üîó Git Merge Commands

git merge branch

Merge branch into current.

Integrates changes from a specified branch into the current branch. Example: git merge feature-x merges feature-x into main
git merge --abort

Abort merge.

Cancels a merge operation and restores the pre-merge state. Example: git merge --abort cancels a conflicting merge.
git merge --squash

Squash commits into one.

Combines a branch's commits into a single commit during merge. Example: git merge --squash feature-x combines feature-x commits into one.

‚¨áÔ∏è Git Pull Commands

git pull

Fetch + merge from remote.

Downloads and integrates changes from a remote repository. Example: git pull updates main with remote changes.
git pull --squash

Pull and squash commits.

Fetches remote changes and combines them into a single commit. Example: git pull --squash combines fetched commits into one.
git pull --no-ff

Force merge commit.

Fetches and merges remote changes, ensuring a merge commit. Example: git pull --no-ff creates a merge commit for pulled changes.

‚¨ÜÔ∏è Git Push Commands

git push

Push changes to remote.

Uploads local branch commits to the remote repository. Example: git push sends local main to remote.
git push remote branch

Push specific branch.

Pushes a specified local branch to a remote repository. Example: git push origin feature-x pushes feature-x to remote.
git push --force

To forcefully overwrite the history of a remote Git branch with the local branch's history.

Overwrites the remote branch with local branch history. Example: git push --force overwrites remote feature-x
git push --tags

Push all tags.

Uploads all local tags to the remote repository. Example: git push --tags sends tags like v1.0 to remote.

üìå Git Rebase Commands

git rebase branch

Rebase onto branch.

Reapplies commits from the current branch onto another branch. Example: git rebase main rebases current branch onto main
git rebase -i branch

Interactive rebase.

Allows interactive editing of commits during rebase. Example: git rebase -i main allows editing commits on current branch.
git rebase --abort

Abort rebase.

Cancels a rebase operation and restores the original state. Example: git rebase --abort cancels a conflicting rebase.
